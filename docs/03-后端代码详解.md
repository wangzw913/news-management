    TOKEN_EXPIRED(1006, "令牌已过期"),

**为什么用枚举？** 统一管理错误码，前后端对错误码的理解一致。前端可以根据 `code` 做不同处理（如 1006 自动刷新 token）。

#### BusinessException.java
```java
public class BusinessException extends RuntimeException {
    private final int code;

    public BusinessException(ErrorCode errorCode) {
        super(errorCode.getMessage());
        this.code = errorCode.getCode();
    }
}
```
**为什么继承 RuntimeException？** Spring 的事务管理默认只回滚 RuntimeException。继承它才能让业务异常触发事务回滚。

#### GlobalExceptionHandler.java
```java
@RestControllerAdvice  // 全局拦截所有 Controller 的异常
public class GlobalExceptionHandler {

    @ExceptionHandler(BusinessException.class)
    public Result<Void> handleBusiness(BusinessException e) {
        return Result.fail(e.getCode(), e.getMessage());
    }

    @ExceptionHandler(MethodArgumentNotValidException.class)
    public Result<Void> handleValidation(MethodArgumentNotValidException e) {
        // 取出所有校验失败的字段和理由
        String msg = e.getBindingResult().getFieldErrors().stream()
            .map(f -> f.getField() + ": " + f.getDefaultMessage())
            .collect(Collectors.joining("; "));
        return Result.badRequest(msg);
    }

    @ExceptionHandler(AccessDeniedException.class)
    public Result<Void> handleAccessDenied() {
        return Result.forbidden("权限不足");
    }

    @ExceptionHandler(Exception.class)  // 兜底
    public Result<Void> handleUnknown(Exception e) {
        log.error("系统异常", e);  // 记录日志，不要把堆栈返回给前端
        return Result.fail("系统内部错误，请联系管理员");
    }
}
```

### 用户模块（modules/user/）

#### UserService.java —— 最核心的业务逻辑

```java
@Service
public class UserService extends ServiceImpl<UserMapper, User> {

    // 登录验证
    public User login(LoginDTO dto) {
        // 1. 查用户（只按用户名查，不再需要选角色）
        User user = this.getOne(new LambdaQueryWrapper<User>()
            .eq(User::getUsername, dto.getUsername()));

        // 2. 验证密码（BCrypt.matches 而不是 ==）
        if (user == null || !passwordEncoder.matches(dto.getPassword(), user.getPassword())) {
            throw new BusinessException(ErrorCode.PASSWORD_ERROR);
        }

        // 3. 检查状态
        if (user.getStatus() == 0) {
            throw new BusinessException(ErrorCode.ACCOUNT_DISABLED);
        }

        return user;
    }

    // 注册（重要安全点！）
    public void register(RegisterDTO dto) {
        // 1. 检查用户名唯一
        if (this.count(new LambdaQueryWrapper<User>()
                .eq(User::getUsername, dto.getUsername())) > 0) {
            throw new BusinessException(ErrorCode.USERNAME_EXISTS);
        }

        // 2. 检查邮箱唯一（如果提供了）
        if (dto.getEmail() != null && !dto.getEmail().isEmpty()) {
            if (this.count(new LambdaQueryWrapper<User>()
                    .eq(User::getEmail, dto.getEmail())) > 0) {
                throw new BusinessException(1007, "该邮箱已被注册");
            }
        }

        // 3. 创建用户
        User user = new User();
        user.setUsername(dto.getUsername());
        user.setPassword(passwordEncoder.encode(dto.getPassword()));  // BCrypt 加密！
        user.setRole("user");   // ← 强制普通用户！不接受前端传来的角色
        user.setEmail(dto.getEmail());
        user.setPhone(dto.getPhone());
        user.setStatus(1);      // 默认启用
        this.save(user);
    }
}
```

**面试关键点**：
- `user.setRole("user")` 强制角色，这是安全设计！防止有人注册时传 `role=admin`
- 登录不再需要选角色（删除了 `.eq(User::getRole, dto.getRole())`），系统自动识别

### 新闻模块（modules/news/）—— 最复杂

#### NewsService.java

```java
@Service
public class NewsService extends ServiceImpl<NewsMapper, News> {

    // ===== 核心流程：审核工作流 =====

    // 状态机：draft → pending → published/rejected

    public void submitForAudit(Long newsId, Long userId) {
        News news = this.getById(newsId);
        // 只有草稿和被退回的稿件可以提交
        if (!"draft".equals(news.getStatus()) && !"rejected".equals(news.getStatus())) {
            throw new BusinessException(ErrorCode.NEWS_STATUS_ERROR);
        }
        news.setStatus("pending");
        news.setRejectReason(null);  // 清除旧的退回原因
        this.updateById(news);
    }

    public void approveNews(List<Long> ids) {
        for (Long id : ids) {
            News news = this.getById(id);
            if (news != null && "pending".equals(news.getStatus())) {
                news.setStatus("published");
                this.updateById(news);
            }
        }
    }

    public void rejectNews(Long newsId, String reason) {
        News news = this.getById(newsId);
        if (news == null || !"pending".equals(news.getStatus())) {
            throw new BusinessException(ErrorCode.NEWS_STATUS_ERROR);
        }
        news.setStatus("rejected");
        news.setRejectReason(reason);
        this.updateById(news);
    }

    public void deleteNews(Long newsId, Long userId, String role) {
        News news = this.getById(newsId);
        // 权限检查：admin 可以删任何人的，编辑只能删自己的
        if (!"admin".equals(role) && !userId.equals(news.getUserId())) {
            throw new BusinessException(ErrorCode.NEWS_PERMISSION_DENIED);
        }
        this.removeById(newsId);
    }

    // ===== 点击计数缓冲 =====
    private final ConcurrentHashMap<Long, Long> clickBuffer = new ConcurrentHashMap<>();

    public void incrementClick(Long newsId) {
        clickBuffer.merge(newsId, 1L, Long::sum);  // 原子操作，线程安全
    }

    @Scheduled(fixedRate = 300000)  // 每 5 分钟执行一次
    public void flushClickCounts() {
        if (clickBuffer.isEmpty()) return;
        Map<Long, Long> snapshot = new HashMap<>(clickBuffer);
        clickBuffer.clear();  // 先清空再写库（防止丢失更新）

        for (Map.Entry<Long, Long> entry : snapshot.entrySet()) {
            News news = this.getById(entry.getKey());
            if (news != null) {
                news.setClicked(news.getClicked() + entry.getValue());
                this.updateById(news);
            }
        }
    }

    @PreDestroy  // 应用关闭前
    public void flushOnShutdown() { flushClickCounts(); }
}
```

**面试关键点**：
- `ConcurrentHashMap`：线程安全的 HashMap，多线程并发修改不丢数据
- `@Scheduled(fixedRate = 300000)`：定时任务，Spring 自动调度
- `@PreDestroy`：JVM 关闭前执行，防止缓冲数据丢失
